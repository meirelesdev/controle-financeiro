# Controle Financeiro

> Aplicacao PWA de controle financeiro pessoal com TypeScript, Vite, Chart.js para graficos e suporte a armazenamento offline com IndexedDB.

## Stack

- TypeScript
- Tailwind CSS
- Vite
- Chart.js

## Prerequisitos

- Node.js v14+

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/controle-financeiro.git
cd controle-financeiro

# Instale as dependencias
npm install

# Inicie a aplicacao
npm run dev
```
